# PixelJoy Draw GitHub Pages

This folder contains a free static website for:

- `Support URL`
- `Privacy Policy URL`

## Quick Publish With GitHub Pages

1. Create a new public GitHub repository.
2. Upload the contents of this `Website` folder to the root of that repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Save and wait a few minutes.
6. Your site URL will look like:
   - `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/`

## URLs to Use in App Store Connect

- Support URL:
  - `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/support.html`
- Privacy Policy URL:
  - `https://YOUR-USERNAME.github.io/REPOSITORY-NAME/privacy.html`

If you later create a custom domain, you can swap these URLs in App Store Connect.
